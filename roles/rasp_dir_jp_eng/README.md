rasp_dir_jp_eng
=========

Configures directory names in Japanese and English for Raspberry Pi

Requirements
------------

- Ansible 2.9 or higher
- Supported operating systems: Linux

Role Variables
--------------

This role has no configurable variables.

Dependencies
------------

None

Example Playbook
----------------

```yaml
- hosts: servers
  become: yes
  roles:
    - rasp_dir_jp_eng
```

----
